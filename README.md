## ✨ Key Features

- **🧹 Anti-Suicide Cleanup Engine:** Optimized to clean User Temp, System Temp, and Prefetch files without crashing the compiled EXE.
- **🚀 System Performance:** Real-time Windows Experience Index (WinSAT) assessment for CPU, RAM, and Disk.
- **🔍 Hardware Health:** S.M.A.R.T. status monitoring for SSD/HDD health and life-left percentage.
- **🌳 System32 Mapping:** Specialized Tree Utility that targets `C:\Windows\System32` for deep directory visualization.
- **📦 WinGet Integration:** Search, install, and batch-upgrade all your Windows apps from one menu.
- **⚡ Power Control:** Instant toggle for Hibernation (reclaim GBs of space) and Network stack resets.
- **🛠️ System Repair:** One-click SFC Scannow and "Reboot to BIOS/UEFI" functionality.
